# iOS Lottery App

Мобильное приложение для участия в лотерее на платформе iOS.

## Технологии

- **Swift 5.9+**
- **SwiftUI** для UI
- **MVVM** архитектура
- **URLSession** для сетевых запросов
- **Keychain** для безопасного хранения токенов
- **UserDefaults** для кэширования данных

## Структура проекта

```
LotteryApp/
├── App/                    # Точка входа приложения
├── Models/                 # Модели данных
├── Services/              # Сетевые сервисы
├── ViewModels/            # ViewModels для MVVM
├── Views/                 # SwiftUI экраны
│   ├── Auth/             # Экран входа и регистрации
│   ├── Home/             # Главный экран с игрой
│   └── Profile/          # Профиль и транзакции
└── Utils/                 # Утилиты и константы
```

## Требования

- iOS 15.0+
- Xcode 15.0+
- Swift 5.9+

## Настройка проекта

1. Откройте проект в Xcode
2. Настройте Team в Signing & Capabilities
3. Для работы с HTTP (не HTTPS) добавьте в Info.plist:

```xml
<key>NSAppTransportSecurity</key>
<dict>
    <key>NSAllowsArbitraryLoads</key>
    <true/>
</dict>
```

## Функционал

- ✅ Регистрация и вход
- ✅ Валидация email и пароля
- ✅ Участие в лотерее
- ✅ Таймер обратного отсчета (30 минут)
- ✅ Автоматическая синхронизация статуса игры

- ✅ История транзакций
- ✅ Редирект на Telegram для пополнения

- ✅ Проверка верификации аккаунта

## API

Базовый URL: `http://94.103.9.172:8081/api`

Все эндпоинты требуют JWT токен в заголовке `Authorization: Bearer <token>` (кроме `/register` и `/login`).

## Безопасность

- JWT токены хранятся в Keychain
- Автоматический logout при 401 ошибке
- Валидация всех пользовательских данных
